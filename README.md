# 📚 Fee Reminder App

A modern Android application for managing tuition fee reminders.

## 🚀 Features

- Add, Edit and Delete Students
- Mark Fee Paid
- Automatic Next Due Date (30 Days)
- Dashboard Statistics
- Search Students
- Batch Filtering
- Excel Bulk Import
- Duplicate Detection
- Daily Notifications using WorkManager
- Room Database
- MVVM Architecture
- Jetpack Compose UI

## 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- Room Database
- WorkManager
- Apache POI
- MVVM Architecture

## 📱 Screens

- Dashboard
- Students
- Student Details
- Settings
